# Sample Project

This is a sample project for demonstrating OMNI's capabilities.

## Files

- `auth.py` - Authentication module with intentional security issues
- `config.yaml` - Configuration file

## Purpose

This project is used to demonstrate:

1. **Security Agent** - Finding vulnerabilities in `auth.py`
2. **Compliance Agent** - Checking for GDPR compliance issues
3. **Coding Agent** - Generating patches to fix issues

## Running the Demo

```bash
# From OMNI root directory
python -m demo.demo_mode --full
```
